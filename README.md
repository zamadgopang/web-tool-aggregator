# 🛠️ ToolKit - Premium Web-Based Tools Aggregator

> **Lightning-fast, feature-rich browser tools that run entirely in your browser. 100% client-side, zero server delays, complete privacy.**

## ✨ Key Features

🚀 **High Performance** - All tools run 100% client-side with zero latency  
🔒 **Privacy First** - Your files never leave your browser  
💎 **Premium Design** - Modern UI with dark/light theme support  
♿ **Fully Accessible** - WCAG 2.1 compliant with keyboard navigation  
📱 **Mobile Responsive** - Works seamlessly on all devices  

## 🛠️ 15+ Available Tools

### Image Tools
- **Image Converter** - Convert to JPG, PNG, WebP, BMP, GIF with compression
- **SVG to PNG** - Scale and convert vector graphics

### Security & Encoding  
- **Base64 Converter** - Encode/decode base64
- **Password Generator** - Secure random passwords
- **Hash Generator** - MD5, SHA1, SHA256, SHA512
- **QR Code Generator** - Create QR codes

### Utilities
- **Color Converter** - HEX, RGB, HSL conversion
- **Unit Converter** - Length, weight, temperature, volume, speed
- **Text Minifier** - Minify HTML, CSS, JavaScript
- **JSON Formatter** - Format, validate, minify JSON

### More (Coming Soon)
- PDF tools (merge, compress, convert)
- Video to GIF converter
- Document converters

## 🚀 Quick Start

```bash
# Clone and setup
git clone https://github.com/zamadgopang/web-tool-aggregator.git
cd web-tool-aggregator
pnpm install

# Development
pnpm dev
# Open http://localhost:3000

# Production build
pnpm build
pnpm start
```
